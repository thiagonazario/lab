# Manage multiple containers

Run a nginx, httpd and mysql
Run all of them --detach (or -d), name them with the --name


nginx -p 80:80
httpd -p 8080:80
mysql -p 3306:3306

when running mysql, use --env or -e to pass
MYSQL_RANDOM_ROOT_PASSWORD=yes

use docker container logs on mysql

after thath, stop all the containers and remove them...

also use:
docker stats
docker inspect
